# チュートリアル

## これはなに？

TexTransTool は [専用テクスチャの導入](/docs/Tutorial/SetupExclusiveTexture) や [直感的な色改変](/docs/Tutorial/SetupSimpleDecal)、[強力なアトラス化](/docs/Tutorial/ReductionTextureMemoryByAtlasing) などが可能な、非破壊テクスチャ改変ツールです！

## インストール

[VRChat Creator Companion](https://vcc.docs.vrchat.com) か [ALCOM](https://vrc-get.anatawa12.com/alcom/) からのインストールを推奨しています。  [パッケージ追加リンク](vcc://vpm/addRepo?url=https%3A%2F%2Fvpm.rs64.net%2Fvpm.json)

TexTransTool はアバター改変用途では [NDMF](https://github.com/bdunderscore/ndmf) が必須です、同時にインストールすることを強く推奨します！  
[ModularAvatar](https://modular-avatar.nadena.dev/ja/) や [AvatarOptimizer](https://vpm.anatawa12.com/avatar-optimizer/ja/) が既に導入されている場合は問題ありません。
